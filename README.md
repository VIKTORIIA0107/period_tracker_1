# My Period Tracker

A Flutter application for tracking menstrual cycles and fertility windows.

## Description

My Period Tracker is a comprehensive menstrual cycle tracking application built with Flutter. It provides an intuitive interface for tracking periods, predicting ovulation, and monitoring fertility windows.

## Features

- Monthly and yearly calendar views
- Period tracking
- Ovulation prediction
- Fertility window calculation
- User-friendly design
- Customizable cycle settings

## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- iOS Simulator/Android Emulator

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/VIKTORIIA0107/my_period_tracker.git
   ```
2. Navigate to project directory
   ```bash
   cd periodtracker_1
   ```
3. Install dependencies
   ```bash
   flutter pub get
   ```
4. Run the application
   ```bash
   flutter run
   ```
